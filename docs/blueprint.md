# **App Name**: Crafted Code

## Core Features:

- Dynamic Hero: Interactive hero section with rotating, benefit-driven headlines to quickly engage visitors.
- Service Showcase: A well-structured 'Services' section with clear icons and concise descriptions that link to detailed service pages.
- Category Identifier: A tool to analyze uploaded text and determine its category, improving site indexing.
- Project Showcase: Display a project gallery, linking to detailed case studies to demonstrate expertise and results.
- Contact Gateway: Ensure easy client contact through a simple form and additional contact information.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to convey professionalism and depth.
- Background color: Very light Lavender (#F5EEF8), providing a soft, uncluttered backdrop.
- Accent color: Electric Violet (#8F00FF) for interactive elements and key highlights.
- Body: 'PT Sans', a versatile and readable humanist sans-serif font, suitable for body text on the 'About' and 'Services' pages. Headline: 'Playfair' for titles; pairing with 'PT Sans' provides both elegance and readability.
- Minimalist icons, with a thin line style in electric violet, to represent different services and capabilities.
- Grid-based layout with generous white space to ensure content is easily scannable.
- Subtle transition animations for page navigation to enhance user experience without being distracting.
